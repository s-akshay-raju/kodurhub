# KodurHub

A local services platform for Kodur town built with HTML, CSS and JavaScript.

## Features
- Movie ticket booking with seat selection
- Theatre-wise show timings
- UPI/Card/Netbanking payment flow
- QR code booking confirmation
- Location-based service discovery

## Pages
- Home — service discovery
- Movies — browse and filter movies by language
- Theatres — view show timings per theatre
- Seats — interactive seat selection layout
- Payment — multi-method payment page
- Confirmation — booking confirmation with QR code

## Tech used
- HTML5, CSS3, Vanilla JavaScript
- sessionStorage for data flow between pages
- QR Server API for QR code generation

## Status
Movies section complete. Food delivery and groceries in progress.
